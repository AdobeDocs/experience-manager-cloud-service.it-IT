# Integrare AEM Assets con le applicazioni a valle {#integrate-dynamic-media-open-apis}

Tutte le [risorse approvate](/help/assets/approve-assets.md) disponibili nell&#39;archivio delle risorse di Experience Manager sono disponibili per la distribuzione alle applicazioni a valle.

È possibile integrare l&#39;interfaccia utente personalizzata con l&#39;archivio di Experience Manager Assets utilizzando le API di ricerca e distribuzione oppure utilizzare Adobe Content Advisor.

![Integrazione con l&#39;archivio AEM Assets](assets/asset-selector-integration.png)

Le API ti consentono di cercare le risorse approvate dall’archivio di AEM Assets e quindi consegnarle alle applicazioni a valle utilizzando un URL di consegna. Per ulteriori informazioni, vedere [API Search](/help/assets/search-assets-api.md) e [Delivery](/help/assets/deliver-assets-apis.md).

Contenuto verificato fornisce un&#39;interfaccia utente che si integra facilmente con il repository [!DNL Experience Manager Assets as a Cloud Service] in modo che sia possibile sfogliare o cercare le risorse digitali approvate disponibili nel repository e utilizzarle nell&#39;esperienza di authoring dell&#39;applicazione. Per ulteriori informazioni, vedere [Contenuto verificato](/help/assets/integrate-adobe-non-adobe-applications.md).
